# Finance Dashboard (HTML, CSS, JavaScript)

## Description

A frontend-based finance dashboard application that allows users to track income and expenses. The system includes role-based access where admins can add and delete transactions, while users can only view data.

## Features

* Dashboard with income, expense, and savings summary
* Add transactions using modal popup
* Transactions table with delete option (admin only)
* Role-based UI control (admin/user)
* Data persistence using localStorage

## Tech Stack

* HTML5
* CSS3
* JavaScript (DOM manipulation, localStorage)

## How to Run

Open index.html using Live Server in VS Code
